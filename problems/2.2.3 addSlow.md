# 2.2.3 addSlow

```java
void addSlow() {
    decrement();
    if (frontIsClear()) {
        walkBack();
        increment();
        walkBack();      
        addSlow();
    }
}

void walkBack() {
    turnAround();
    while (frontIsClear()) {
        moveForward();
    }
    if (!leftIsClear()) {
        turnRight();
        moveForward();
        turnRight(); 
    } else {
        turnLeft();
        moveForward();
        turnLeft();
    }
}
```
