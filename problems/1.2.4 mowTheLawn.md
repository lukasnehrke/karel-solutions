# 1.2.4 mawTheLawn

```java
void mowTheLawn() {
    repeat (2) {
        mow();
        turnUpLeft();
        mow();
        turnUpRight();
    }
    mow();
    turnUpLeft();
    mow();
}

void mow() {
    repeat (6) {
        moveForward();
        pickBeeper();
    }
    moveForward();
}

void turnUpLeft() {
    turnLeft();
    moveForward();
    turnLeft();
}

void turnUpRight() {
    turnRight();
    moveForward();
    turnRight();
}
```
